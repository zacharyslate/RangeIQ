# Sensor Network Module

This package defines station and reading schemas, validation rules, station status logic, lightweight calibration hooks, and mock station generation for the RangeIQ sensor-network MVP.

Current scope:

- mock station registry and hourly readings
- validation and station health classification
- registry/config placeholders for future ranch deployments

Future scope:

- real field calibration profiles
- station commissioning workflows
- per-sensor metadata and maintenance history
